开发1个windows桌面应用，名字叫Everypic，python语言开发，特性描述如下：
一、功能：
1、输入关键字，则搜索本地所有图片中包含该关键字内容的图片，例如输入关键字“天空”，则搜索本地所有图片中，包含太空的图片。
2、点击搜索结果的图片文件，可以打开该图片
3、程序第一次运行时，扫描本地所有图片，简历索引库。索引库的信息放入Sqllite数据库，包括如下字段：序号、文件名、文件md5摘要、文件路径、文件图片内容描述、文件大小、创建时间、修改时间、保留字段1、保留字段2
二、前端UI界面：
1、一级菜单：文件(F)、编辑（E）、工具（T）、帮助（H）
2、二级菜单：
文件（F）包含：新建、退出，
编辑（E）包含：剪切、复制、粘贴，
工具（T）包含：选项，
帮助（H）包含：帮助、关于
